<h1 align="center" id="title">Storefront Django</h1>

<p id="description">A Django project demonstrating advanced admin customization and a reusable generic tagging system using Django ContentTypes.</p>

  
  
<h2>üßê Features</h2>

Here're some of the project's best features:

*   Products & Customers management
*   Generic tags attachable to any model
*   Inline tag editing in Django Admin
*   Admin inheritance & re-registration
*   Optimized ORM usage
